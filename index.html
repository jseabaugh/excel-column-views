Features
Everything you need

Named Views

Create as many views as you need per worksheet, each with its own column visibility configuration.

One-click switching

Apply any saved view instantly. The active view is clearly indicated so you always know where you are.

Persists with workbook

Views are saved inside your Excel file. Share the workbook and your views travel with it.

Flexible header row

Your headers aren't always in row 1. Tell Column Views which row to read names from.

How it works
Simple by design
1

Open the Column Views panel

Click the Column Views button in your Excel ribbon to open the task pane.

2

Create a new view

Give it a name, set your header row, then check or uncheck columns to show or hide them.

3

Save and apply

Save the view and apply it with one click. Create as many views as you need.

4

Switch anytime

Jump between views instantly. Use Show All to restore all columns at any time.

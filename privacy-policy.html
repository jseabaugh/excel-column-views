Column Views

Privacy Policy

Last updated: May 2026

Column Views does not collect, store, transmit, or share any personal data. Your data never leaves your device or your own workbook.

Overview

Column Views is a Microsoft Excel add-in that allows users to create and manage named column visibility configurations within their spreadsheets. This privacy policy explains what data the add-in handles and how it is used.

Data We Do Not Collect

Column Views does not collect any of the following:

  • Personal information of any kind (name, email, address, etc.)
  • Spreadsheet data, cell values, or workbook contents
  • Usage analytics or telemetry
  • Device information or identifiers
  • Location data
  • Authentication credentials

How Your Data Is Stored

Column Views saves your view configurations (view names and column visibility settings) directly inside your Excel workbook using Excel's built-in Custom XML Parts storage. This means:

  • Your view configurations are stored only within the workbook file itself
  • No data is sent to any external server
  • No data is stored in any cloud service operated by the add-in developer
  • Your data is under your complete control at all times

Network Requests

When you open the Column Views task pane, Excel loads the add-in's interface files (HTML, CSS, and JavaScript) from GitHub Pages at https://jseabaugh.github.io/excel-column-views/. These are static files only. No user data is included in or transmitted during this request.

The add-in also loads the Office.js library from Microsoft's CDN (https://appsforoffice.microsoft.com). This is a standard Microsoft library used by all Office add-ins.

Third-Party Services

Column Views uses the following third-party services solely to deliver its interface:

  • GitHub Pages — hosts the static add-in files. GitHub's privacy policy applies to these requests.
  • Google Fonts — loads the DM Sans and DM Mono typefaces used in the interface. Google's privacy policy applies to font requests.
  • Microsoft Office.js CDN — delivers the Office JavaScript API library. Microsoft's privacy policy applies.

None of these services receive any of your spreadsheet data or personal information as part of normal add-in operation.

Children's Privacy

Column Views does not knowingly collect any information from children under the age of 13. The add-in collects no personal data from any user of any age.

Changes to This Policy

If this privacy policy changes, the updated version will be posted at this URL with a revised "Last updated" date.

Contact

If you have any questions about this privacy policy, you can open an issue on the GitHub repository.
